# Typing Speed Test ⌨️

![Preview](./preview.jpg)

A typing speed test app built as a Frontend Mentor challenge. Test your WPM, accuracy, and track your personal best.

## 🔗 Links

- [Live Demo](#) <!-- Agrega tu URL de Netlify/GitHub Pages -->
- [Repository](https://github.com/emijauregui/typing-speed_EmiJJN)

## ✨ Features

- Start test by clicking or typing directly on the passage
- Difficulty levels: Easy, Medium, Hard
- Two modes: Timed (60s) and Passage mode
- Real-time WPM, accuracy, and time stats
- Visual feedback: correct (green), errors (red/underlined)
- Personal best persists across sessions
- "High Score Smashed!" celebration with confetti 🎉
- Fully responsive design

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (Vanilla)

## 🧠 What I Learned

- DOM manipulation for real-time character tracking
- localStorage for data persistence
- Timer logic for countdown and count-up modes
- Calculating WPM and accuracy in real time
